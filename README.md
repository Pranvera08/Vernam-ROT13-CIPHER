# Vernam-ROT13-CIPHER
Vernam-ROT13-CIPHER është një projekt Python që implementon dy algoritme klasike të enkriptimit: Vernam Cipher dhe ROT13. Përdoruesi mund të zgjedhë mes enkriptimit dhe dekriptimit për secilin algoritëm përmes një menuje interaktive. Ky projekt synon të demonstrojë bazat e kriptografisë klasike dhe praktikën e programimit modular në Python.
